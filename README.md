# Viper

<p align="center">
  <img src="https://images2.imgbox.com/8f/99/mftQy3Rw_o.png" alt="Viper Logo" width="400"/>
</p>

Viper is an ArchLinux distro, that is in current development, designed to be part of the Viper echosystem which is also in current development, as time progress and the developement advances more documentation will be added to it, the main idea is to make
the distro suitable for workstation and gaming use.


Download the first BETA on sourceforge:
https://sourceforge.net/projects/viper/files/viper-beta-iso/

NOTE: the ISO includes apps, libraries and dependencies for gaming alrady, please have patience while we work on the system.
